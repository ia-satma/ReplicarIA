# Tesis sobre Materialidad de Operaciones (Art. 69-B CFF)

## @TESIS_MAT_001 - Carga Probatoria del Cliente

### [CRITERIO JUDICIAL]

**OPERACIONES INEXISTENTES. CARGA DE LA PRUEBA PARA ACREDITAR MATERIALIDAD 
CORRESPONDE AL CONTRIBUYENTE QUE DIO EFECTOS FISCALES A LOS CFDI.**

Cuando un proveedor es incluido en la lista definitiva de contribuyentes 
que facturan operaciones inexistentes (Art. 69-B, párrafo cuarto del CFF), 
el contribuyente que haya dado efectos fiscales a los comprobantes expedidos 
por dicho proveedor tiene la carga de demostrar que efectivamente adquirió 
los bienes o recibió los servicios amparados en dichos comprobantes.

**Plazo:** 30 días hábiles siguientes a la publicación en el DOF.

**Alternativas:**
1. Acreditar la materialidad de las operaciones, O
2. Corregir su situación fiscal (autocorrección)

## @TESIS_MAT_002 - Elementos de Prueba

### [CRITERIO JUDICIAL]

**MATERIALIDAD DE OPERACIONES. ELEMENTOS QUE EL CONTRIBUYENTE DEBE ACREDITAR 
PARA DESVIRTUAR LA PRESUNCIÓN DE INEXISTENCIA.**

Para acreditar la materialidad de operaciones realizadas con proveedores 
incluidos en la lista del artículo 69-B del CFF, el contribuyente debe 
demostrar:

1. **Capacidad del proveedor**
   - Que contaba con personal para prestar el servicio
   - Que tenía infraestructura necesaria
   - Que su domicilio era localizable
   - Que tenía activos relacionados con la actividad

2. **Realidad de la operación**
   - Existencia de contrato o acuerdo
   - Entregables recibidos y documentados
   - Comunicaciones entre las partes
   - Evidencia del servicio prestado

3. **Flujo financiero**
   - Pagos mediante transferencia bancaria
   - Estados de cuenta que acrediten el pago
   - Conciliación del pago con el CFDI

4. **Razonabilidad económica**
   - Precio acorde al mercado
   - Necesidad del servicio para el negocio
   - Relación comercial justificada

## @TESIS_MAT_003 - Documentación Idónea

### [CRITERIO JUDICIAL]

**MATERIALIDAD. LA DOCUMENTACIÓN PARA ACREDITARLA DEBE SER IDÓNEA, 
SUFICIENTE Y CONGRUENTE CON LA NATURALEZA DE LAS OPERACIONES.**

No basta con presentar el CFDI y el comprobante de pago para acreditar 
la materialidad de una operación. La documentación debe ser:

- **Idónea:** Apropiada para demostrar el tipo de operación
- **Suficiente:** Completa para acreditar todos los elementos
- **Congruente:** Consistente entre sí y con la operación declarada

**Ejemplos de documentación idónea por tipo de servicio:**

| Tipo de servicio | Documentación esperada |
|-----------------|------------------------|
| Consultoría | Contrato, informes, minutas, entregables |
| Capacitación | Lista de asistencia, material, constancias |
| Mantenimiento | Bitácoras, fotografías, reportes técnicos |
| Construcción | Planos, permisos, avances de obra, actas |
| Publicidad | Artes finales, pautas, evidencia de difusión |

## [INTERPRETACIÓN REVISAR-IA]

### Proceso cuando proveedor aparece en lista 69-B:

```
1. SAT publica proveedor en lista PRESUNTO
         ↓
2. Proveedor tiene 15 días para desvirtuar
         ↓
3. Si no desvirtúa → Lista DEFINITIVO
         ↓
4. CLIENTE tiene 30 días hábiles para:
   ├── Opción A: Acreditar materialidad
   │   └── Presentar pruebas ante SAT
   └── Opción B: Autocorregirse
       └── Presentar declaraciones complementarias
```

### Checklist de materialidad A7 para Defense File:

```
CAPACIDAD DEL PROVEEDOR
□ Verificación de RFC activo (al momento de operación)
□ Opinión de cumplimiento 32-D
□ Constancia de situación fiscal
□ Evidencia de personal/infraestructura

REALIDAD DE LA OPERACIÓN
□ Contrato firmado con alcance detallado
□ Órdenes de servicio/compra
□ Entregables recibidos (informes, productos)
□ Correspondencia (emails, mensajes)
□ Fotografías/videos (si aplica)
□ Actas de entrega-recepción

FLUJO FINANCIERO
□ CFDI válido (verificado en SAT)
□ Comprobante de pago (transferencia)
□ Estado de cuenta bancario
□ Conciliación pago-factura
□ Complemento de pago (si PPD)

RAZONABILIDAD
□ Cotizaciones comparativas
□ Justificación de la necesidad
□ Relación precio-mercado
□ Historial comercial (si existe)
```

### Niveles de documentación por riesgo:

| Riesgo del proveedor | Documentación requerida |
|---------------------|------------------------|
| No publicado | Básica (contrato, pago, entregable) |
| Presunto | Completa + verificación capacidad |
| Definitivo | Exhaustiva + expediente formal |
| Sentencia favorable | Copia de la sentencia |

### Errores comunes que debilitan la defensa:

1. **Presentar solo CFDI y pago**
   - Insuficiente para acreditar materialidad
   - Se requiere evidencia de la operación real

2. **Contratos genéricos**
   - Deben especificar alcance, entregables, fechas
   - Evitar formatos tipo sin personalización

3. **Sin entregables documentados**
   - Todo servicio debe generar un producto verificable
   - Minutas, informes, productos, etc.

4. **Comunicaciones inexistentes**
   - Operaciones reales generan correspondencia
   - Conservar emails, mensajes, notas

### Ejemplo de expediente bien documentado:

```
EXPEDIENTE DE MATERIALIDAD
Proveedor: SERVICIOS XYZ SA DE CV
RFC: SXY123456789
Status 69-B: PRESUNTO (publicado: 15-ene-2026)

1. CAPACIDAD DEL PROVEEDOR
   ✓ RFC activo al momento de operación
   ✓ Opinión 32-D positiva vigente
   ✓ Domicilio verificable
   ✓ Personal: 12 empleados (constancia IMSS)
   
2. CONTRATO
   ✓ Contrato firmado: 01-mar-2025
   ✓ Vigencia: 12 meses
   ✓ Alcance: Servicios de consultoría fiscal
   ✓ Entregables: Diagnóstico fiscal mensual
   ✓ Monto: $50,000 + IVA mensual
   
3. ENTREGABLES
   ✓ 12 informes mensuales (adjuntos)
   ✓ 4 reuniones de seguimiento (minutas)
   ✓ Correos de coordinación (57 emails)
   
4. PAGOS
   ✓ 12 CFDIs (UUIDs listados)
   ✓ 12 transferencias bancarias
   ✓ Estados de cuenta de Ene-Dic 2025
   
5. RAZONABILIDAD
   ✓ Cotización de 2 proveedores alternativos
   ✓ Precio dentro de rango de mercado
   ✓ Necesidad: Preparación para auditoría SAT

CONCLUSIÓN: MATERIALIDAD ACREDITADA
```

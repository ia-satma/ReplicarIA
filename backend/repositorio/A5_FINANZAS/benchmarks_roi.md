## 7. Validación de Precio de Mercado

### Proceso de Validación
1. Identificar tipo de servicio
2. Consultar benchmark correspondiente
3. Aplicar factores de ajuste
4. Calcular rango esperado
5. Comparar con precio cotizado

### Fórmula de Rango Aceptable
```
Precio Mínimo = Benchmark Base × (1 - 20%)
Precio Máximo = Benchmark Base × (1 + Factor Ajuste)
```

### Alertas por Desviación
| Desviación | Alerta | Acción |
|------------|--------|--------|
| < -30% | Alta | Verificar calidad/capacidad |
| -30% a -10% | Media | Revisar alcance |
| -10% a +20% | Normal | Aprobar |
| +20% a +50% | Media | Justificar |
| > +50% | Alta | Rechazar o renegociar |

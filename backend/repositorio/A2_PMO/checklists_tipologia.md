# Checklists por Tipología de Servicio

---

## Tipología 1: Consultoría Estratégica

### Documentos Requeridos
- [ ] Contrato de servicios profesionales
- [ ] Orden de trabajo o alcance detallado
- [ ] Propuesta técnica aprobada
- [ ] Cronograma de actividades
- [ ] Reportes de avance (semanales/mensuales)
- [ ] Minutas de reunión
- [ ] Entregables finales
- [ ] Acta de entrega y satisfacción
- [ ] Factura CFDI

### Evidencias de Materialidad
- [ ] Presentaciones entregadas
- [ ] Documentos de análisis
- [ ] Recomendaciones formales
- [ ] Correos de validación del cliente

## Checklist Universal de Validación Fiscal

### Para CUALQUIER tipología, verificar:
- [ ] RFC del proveedor válido (no en 69-B)
- [ ] Opinión de cumplimiento positiva (32-D)
- [ ] Actividad económica compatible
- [ ] Sustancia económica demostrable
- [ ] Razón de negocios clara
- [ ] Precio de mercado justificable
- [ ] Trazabilidad documental completa

---
*Usar como guía para validación de expedientes.*

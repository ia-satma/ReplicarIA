# Guía de Entregables por Tipo de Proveedor

## Documentación Requerida para Acreditar Materialidad

---

## 1. Principio General

**Todo servicio debe tener entregables tangibles que demuestren:**
- Que el trabajo se realizó efectivamente
- Que existe un producto o resultado del servicio
- Que el precio pagado corresponde al valor recibido

---

## 2. Consultoría Estratégica

### Entregables Esperados
| Fase | Entregable | Formato |
|------|------------|---------|
| Diagnóstico | Informe de situación actual | PDF/PPTX |
| Análisis | Documento de hallazgos | PDF |
| Recomendaciones | Plan de acción | PDF/PPTX |
| Implementación | Guías y procedimientos | DOCX/PDF |
| Cierre | Informe final | PDF |

### Evidencias Complementarias
- [ ] Minutas de reuniones firmadas
- [ ] Presentaciones entregadas
- [ ] Correos de validación del cliente
- [ ] Fotografías de sesiones de trabajo
- [ ] Listas de asistencia a talleres

### Checklist de Materialidad
- [ ] Documento entregado con logo del consultor
- [ ] Fecha de elaboración dentro del período contratado
- [ ] Contenido específico para el cliente (no genérico)
- [ ] Firmado o validado por el cliente
- [ ] Referenciado en el CFDI

- [ ] Archivos entregados en formato editable
- [ ] Derechos de uso transferidos
- [ ] Evidencia de proceso creativo
- [ ] Publicación/uso del material verificable
- [ ] Brief original vs entregable final

## 8. Formato Universal de Acta de Entrega

```markdown
# ACTA DE ENTREGA DE SERVICIOS

**Fecha:** {{FECHA}}
**Número de Contrato:** {{NUM_CONTRATO}}
**Proveedor:** {{NOMBRE_PROVEEDOR}} - RFC: {{RFC}}
**Cliente:** {{NOMBRE_CLIENTE}}

## Servicios Contratados
{{DESCRIPCION_SERVICIOS}}

## Entregables Recibidos

| # | Entregable | Descripción | Recibido |
|---|------------|-------------|----------|
| 1 | {{ENTREGABLE_1}} | {{DESC_1}} | ☑️ |
| 2 | {{ENTREGABLE_2}} | {{DESC_2}} | ☑️ |

## Observaciones
{{OBSERVACIONES}}

## Conformidad
El cliente manifiesta su conformidad con los servicios recibidos.

___________________________
Firma del Cliente
{{NOMBRE_REPRESENTANTE}}

___________________________
Firma del Proveedor
{{NOMBRE_PROVEEDOR}}
```

## 9. Matriz de Entregables Mínimos por Monto

| Monto (MXN) | Entregables Mínimos |
|-------------|---------------------|
| < $50,000 | Factura + 1 evidencia de trabajo |
| $50,000 - $200,000 | Factura + Contrato + Entregables documentados |
| $200,000 - $500,000 | Todo anterior + Acta de entrega + Evidencias fotográficas |
| > $500,000 | Todo anterior + Bitácora completa + Validación de tercero |
